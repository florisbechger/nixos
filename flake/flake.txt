## Provision Remote on Workstation (https://galowicz.de/2023/04/05/single-command-server-bootstrap):
nix flake show
nix flake update

nix run github:nix-community/nixos-anywhere -- admin@192.168.122.x --flake .#vm1

# Regenerate hardware-configuration.nix on Remote system, usefull for external hardware:
nixos-generate-config --show-hardware-config > /etc/nixos/hardware-configuration.nix
# or, during Provisioning:
nix run github:nix-community/nixos-anywhere -- --generate-hardware-config ./hardware-configuration.nix --nixos-generate-config>

# Update Remote system with update flake.nix:
nixos-rebuild switch --flake .#vm1 --target-host root@192.168.122.x
